# Prof Informatique - AI-Powered CS Education Platform

This is a modern educational platform for computer science, built with React, TypeScript, and Vite. It leverages Google's Gemini API to provide a suite of AI-powered tools for students.

## Features

- **Dashboard:** Overview of progress and stats.
- **Interactive Lessons:** Engaging course material.
- **Exercises:** Test your knowledge.
- **AI Tools:**
  - **Advanced Assistant:** Get help with complex topics.
  - **Image Analyzer:** Analyze diagrams and code snippets from images.
  - **Image Generator:** Create visualizations and diagrams.
  - **Image Editor:** Modify existing images with AI.
  - **Video Generator:** Create educational videos from prompts.

## Tech Stack

- **Framework:** React 19.2.0 & TypeScript 5.8.2
- **Build Tool:** Vite 6.2.0
- **Styling:** Tailwind CSS
- **AI:** @google/genai 1.28.0
- **Charts:** Recharts 3.3.0
